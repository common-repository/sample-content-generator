This plugin generate sample html content in WordPress editor.
This plugin use LoremIpsum.net API, http://loripsum.net/api and http://dummyimage.com/.

== Description ==

This plugin add a tiny MCE button for generating sample html content.

== Screenshots ==

1. Options for generating content

== Installation ==

1. Upload `/sample-content-generator/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to add page/post and click on "Generate Content" above the editor
